Teamtalk Type Linking
  • Select a Teamtalk event type from the left hand column and a normal event from the right hand list and click Link/Unlink to link it.
  • Highlight a linked Teamtalk event, and click 'Link/Unlink to unllink it.
  • Highlight a linked Teamtalk event, and click Generate to generate links at race and selection level
